# 🚀 TaskFlow Management System

A full-stack MERN Team Task Management application with role-based access control, project management, task assignment, dashboard analytics, and production deployment.

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/mohammedfaizdev-prog/taskflow-management-system.git
```

---

## 2️⃣ Backend Setup

```bash
cd server
npm install
npm run dev
```

Create `.env`

```env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
```

---

## 3️⃣ Frontend Setup

```bash
cd client
npm install
npm run dev
```
